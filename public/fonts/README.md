# Font Dosyaları

Bu klasöre aşağıdaki font dosyalarını eklemeniz gerekmektedir:

1. **Freshman.ttf** veya **Freshman.otf** - Uygulama adı için
   - İndirme: https://www.cdnfonts.com/tr/freshman.font
   - Dosya adı: Freshman.ttf veya Freshman.otf

2. **Basketball.ttf** veya **Basketball.otf** - Tarih bilgileri için
   - İndirme: https://www.dafont.com/basketball.font
   - Dosya adı: Basketball.ttf veya Basketball.otf

Font dosyalarını bu klasöre ekledikten sonra uygulama otomatik olarak bu fontları kullanacaktır.
